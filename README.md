# 🛒 Amazon E-Commerce Website

A frontend e-commerce website inspired by Amazon, developed to demonstrate web development skills using HTML, CSS, and JavaScript. The project includes product browsing, search, category filtering, sorting, pagination, shopping cart management, and order confirmation.

## 📌 Project Overview

This project is a responsive e-commerce website that allows users to browse products, search for products, filter products by category, sort products based on price or name, and manage items in a shopping cart.

The project was developed as a practical frontend web development project to improve skills in HTML, CSS, JavaScript, JSON, DOM manipulation, and browser local storage.

## ✨ Features

* 🛍️ Product listing
* 🔍 Product search
* 📂 Category filtering

## ⚙️ How It Works

### Product Browsing

Users can browse available products displayed on the home page.

### Search and Filtering

Users can search for products by name and filter products based on categories.

### Sorting

Products can be sorted by:

* Price: Low to High
* Price: High to Low
* Name: A to Z
* Name: Z to A

### Shopping Cart

Users can add products to the cart and manage product quantities using the increase and decrease buttons.

### Local Storage

The shopping cart uses browser Local Storage to preserve cart data during the browsing session.

### Order Confirmation

After placing an order, the user is redirected to an order confirmation page.
